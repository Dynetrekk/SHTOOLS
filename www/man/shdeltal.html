SHDeltaL

       SHDeltaL - Compute the spectral-concentration criterion of Narcowich
                  and Ward (1996).

SYNOPSIS

       REAL*8 FUNCTION SHDeltaL ( COEF, LMAX )

           REAL*8    COEF(LMAX+1)

           INTEGER   LMAX

DESCRIPTION

       SHDeltaL will calculate the spectral concentration criterion of
       Narcowich and Ward (1996) for a window that possesses non-zero
       coefficients for only a single order M. This is given explicitly by

       Sum_{l=0}^LMAX l * (l+1) COEF(l+1)**2.

       The input coefficients must correspond to either geodesy-normalized or
       orthonormalized spherical harmonics.

ARGUMENTS

       COEF         (input) REAL*8, DIMENSION (LMAX+1)

                    The geodesy-normalized or orthonormalized coefficients of
                    a window which has non-zero coefficients for only a single
                    order M. The actual order in unimportant for this routine.

       LMAX         (input) INTEGER

                    The maximum spherical harmonic degree of the input window
                    coefficients.

REFERENCES

       Narcowich, F.J. and J.D. Ward, Nonstationary wavelets on the m-sphere
       for scattered data, App. Comput. Harm. Anal., 3, 324-336, 1996.

       Wieczorek, M. A. and F. J. Simons, Localized spectral analysis on the
       sphere, Geophys. J. Int., 162, 655-675.
